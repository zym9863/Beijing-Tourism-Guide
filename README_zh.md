# 北京旅游指南

[English](README.md) | [中文](README_zh.md)

这是一个为游客提供的五一假期北京旅游综合性网页指南。该交互式指南提供了详细的4天行程安排，包含景点信息、交通指引、天气预报、餐饮推荐和实用旅行贴士。

## 功能特点

- **详细的每日行程安排**：精心规划的4天行程，涵盖北京最具代表性的景点
- **天气信息**：每日天气预报，帮助旅行者做好适当准备
- **交通指引**：使用公共交通在景点之间移动的详细信息
- **餐饮推荐**：每个景点附近的用餐建议，特色当地美食
- **实用旅行贴士**：关于导航北京的有用建议，包括支付方式、订票和当地习俗
- **响应式设计**：适用于各种设备的移动友好布局

## 使用的技术

- HTML5
- CSS3（使用自定义变量和动画）
- Font Awesome 图标
- 响应式设计技术

## 预览

该指南包括以下部分：
- 行程概览（4天行程）
- 详细的每日计划，包含：
  - 天气预报
  - 上午和下午的活动
  - 交通信息
  - 景点详情，包括参观时间和票价
  - 餐饮推荐
  - 每日贴士
- 实用信息（交通、住宿、美食、购物）
- 北京旅游的一般贴士

## 使用方法

1. 克隆或下载此仓库
2. 在任何现代网页浏览器中打开 `index.html`
3. 浏览指南，规划您的北京之旅
