# Beijing Tourism Guide

[English](README.md) | [中文](README_zh.md)

A comprehensive web-based travel guide for tourists planning to visit Beijing during the May Day holiday. This interactive guide provides a detailed 4-day itinerary with information about attractions, transportation, weather forecasts, dining recommendations, and practical travel tips.

## Features

- **Detailed Day-by-Day Itinerary**: A carefully planned 4-day schedule covering Beijing's most iconic attractions
- **Weather Information**: Daily weather forecasts to help travelers prepare appropriately
- **Transportation Guidance**: Detailed information on how to get between attractions using public transportation
- **Dining Recommendations**: Suggestions for meals near each attraction, featuring local cuisine
- **Practical Travel Tips**: Useful advice for navigating Beijing, including payment methods, booking tickets, and local customs
- **Responsive Design**: Mobile-friendly layout that works on various devices

## Technologies Used

- HTML5
- CSS3 (with custom variables and animations)
- Font Awesome icons
- Responsive design techniques

## Preview

The guide includes the following sections:
- Trip overview (4-day itinerary)
- Detailed day plans with:
  - Weather forecasts
  - Morning and afternoon activities
  - Transportation information
  - Attraction details with visiting times and ticket prices
  - Meal recommendations
  - Daily tips
- Practical information (transportation, accommodation, food, shopping)
- General travel tips for Beijing

## Usage

1. Clone or download this repository
2. Open `index.html` in any modern web browser
3. Navigate through the guide to plan your Beijing trip
